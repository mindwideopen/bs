# Baltic Style landing

Static Next.js landing page for Baltic Style, built with App Router, TypeScript,
styled-components, and Feature-Sliced Design.

## Scripts

```bash
pnpm dev
pnpm lint
pnpm build
pnpm run deploy
```

The production build is exported to `out/` and deployed to the `gh-pages` branch
root. GitHub Pages should use `gh-pages / root` as its source.
